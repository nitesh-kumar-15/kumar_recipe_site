Project Responsibilities (Solo)

All work for Quick Weeknight Recipe Studio was completed by Nitesh Kumar. Key areas included:

Structure & Content

  • Semantic HTML5 across 11 pages (index, recipes, about, team, 7 recipe details)
  • Recipe content and copy for each dish (ingredients, steps, tips)
  • ARIA labels, landmarks, and logical heading order

Layout & Styling

  • CSS variables for colors, typography, spacing, and shadows
  • Flexbox for header/nav; Grid for recipe cards and detail two-column layout
  • Reusable card, chip, and hero image components
  • Responsive behavior at 768px (stacked layout, nav reflow)

Motion & Accessibility

  • Keyframe animations (fadeIn, slideIn) and card hover transitions
  • Story Pack :target behavior for recipe section highlighting
  • Visible focus states and prefers-reduced-motion support
